## GitHub Pages

1. Push this project to your GitHub repo.
2. Go to Settings ‚Üí Pages.
3. Choose Deploy from branch.
4. Select main and root.
5. Add custom domain: `ormamolta.is`.
6. Enable HTTPS when GitHub allows it.
